OVERVIEW OF PROJECT:

Project : Library Management System

Objective :The primary objective of this project is to build a lightweight, robust, and interactive tool for managing library inventory while reinforcing core foundational patterns in structural programming.
Technical Learning Objectives:
* **Structured Data Modeling:** Learn how to define and utilize C structures (`struct`) to encapsulate diverse data points (ID, text strings, and status flags) into a single cohesive object.
* **State Management:** Implement operational state logic (using an integer flag as a boolean indicator) to accurately track and toggle the real-time availability of items ("Issued" vs. "Available").
* **Streamlined Text Processing:** Gain hands-on experience with safe text inputs via `fgets()`, parsing multi-word strings, and cleaning data strings using string library functions like `strcspn()`.
* **Defensive Input Validation:** Write code that checks boundaries (e.g., stopping additions when the database is full) and filters out structural collisions like matching duplicate item IDs.
* **Modular Logic Separation:** Practice breaking down application logic into clean, distinct, single-purpose functions that handle individual runtime scenarios cleanly.
